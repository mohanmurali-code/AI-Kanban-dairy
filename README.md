## Planned stack
- UI: React + TypeScript + Vite
- Desktop: Tauri (Windows-friendly) or Electron
- Drag & Drop: dnd-kit
- Editor: Tiptap or Slate
- State: Zustand or Redux Toolkit

See docs/Roadmap.md for milestones and next actions.
